# 个人网站搭建计划

## 技术选型

### 前端
- **Next.js 14** - React框架，SEO友好
- **TailwindCSS** - 样式框架
- **shadcn/ui** - UI组件库

### 后端
- **Next.js API Routes** - 轻量级后端
- **你的服务器** - 腾讯云部署

### 内容管理
- **Markdown** - 博客文章
- **JSON** - 数据存储（初期）

## 网站结构

```
/
├── 首页
│   ├── Hero区域 - 个人介绍
│   ├── 视频展示 - 最新内容
│   ├── 技能展示 - Java/AI/自动化
│   └── 联系方式
├── /blog
│   ├── 博客文章列表
│   └── 文章详情页
├── /projects
│   ├── 副业项目展示
│   └── 项目详情
└── /contact
    └── 联系表单
```

## 核心功能

1. **个人品牌展示**
   - 头像、简介、技能标签
   - 玻璃拟态风格

2. **内容展示**
   - 视频卡片
   - 博客文章
   - 项目案例

3. **互动功能**
   - 联系表单
   - 评论系统（后期）
   - 订阅通知

4. **SEO优化**
   - 元数据配置
   - sitemap生成
   - 结构化数据

## 设计风格

- **色调**: 深色模式 + 渐变
- **风格**: 现代简约、科技感
- **动画**: 微交互、平滑过渡
