# 从0到1：AI教我做自媒体 - 视频脚本

## 项目说明
- 主题：真实的学习过程记录
- 手段：每个技能都由AI教
- 录制方式：纯声音出镜

## 第1期：我决定做自媒体博主，但完全不会

### 标题片（4秒）
- 文字：想做自媒体，但完全不会...
- 背景：迷茫感
- 音乐：略带疑问的音效

### 第一幕：现状（18秒）
- 画面：对着镜头（或手机自拍）的自述
  - 表情：略带迷茫
  - 手势：摊手、摇头
- 文字：想做自媒体博主，但完全不会
- 旁白："我做了5年Java开发，一直想做自媒体博主。VS Code里用Copilot写代码没问题，Trae也能写代码。但每次想到要录视频、剪辑、做封面，就放弃了。因为...我完全不会。"

### 第二幕：具体困难（25秒）
- 画面：列举困难的快速切换
  1. "我不会录屏"（摇头）
  2. "我不会剪辑"（摊手）
  3. "我不会做封面"（皱眉）
  4. "我不懂镜头语言"（看旁边）
- 文字：每一步都是拦路虎
- 旁白："我不会录屏，不会用OBS。我不会剪辑，不会用PR、剪映。我不会做封面，不会PS。甚至面对镜头，我都不知道说什么。每一步都是拦路虎。"

### 第三幕：转机（25秒）
- 画面：突然想到的表情
  - 眼神一亮
  - 微笑
- 文字：等等...这些都可以用AI做？
- 旁白："但今天我突然想到：现在是AI时代啊！我写代码都用AI了，为什么做视频不行？录视频、剪辑、封面、文案，这些不都是AI能做的吗？"

### 第四幕：决定（18秒）
- 画面：坚定的表情
  - 握拳
  - 点头
- 文字：从今天开始，我公开记录从0到1的过程
- 旁白："我决定做一件事：从今天开始，我公开记录自己从0到1做自媒体博主的过程。每学一个新技能，就做一期视频。用AI教我做自媒体。"

### 第五幕：预告（15秒）
- 画面：列出后续计划
  1. AI教我录屏
  2. AI教我剪辑
  3. AI教我做封面
  4. AI教我写脚本
- 文字：下期：AI教我录屏
- 旁白："下期视频，我会用AI学习怎么录屏。如果你也想做自媒体但完全不会，关注我，我们一起学。"

### 尾片（5秒）
- 文字：从0到1，记录真实的成长
- 背景：频道LOGO

## 第2期：AI教我录屏

### 标题片（4秒）
- 文字：完全不会录屏...
- 背景：困惑感
- 音乐：探索感音效

### 第一幕：问题（15秒）
- 画面：手机自述
  - 表情：困惑
  - 手势：摊手
- 文字：完全不会录屏，OBS都不知道怎么用
- 旁白："上期视频我说要做自媒体，但完全不会录制。OBS听说过，但从来没打开过。录屏到底要怎么弄？"

### 第二幕：向OpenClau求助（20秒）
- 画面：打开OpenClau对话框
  - 输入：OpenClau，教我如何用OBS录屏
  - 等待：AI回复
- 文字：用OpenClau问AI
- 旁白："我想起正在用的OpenClau，它不是什么都能教吗？于是我问它：OpenClau，教我如何用OBS录屏。"

### 第三幕：AI详细指导（30秒）
- 画面：OpenClau回复的滚动
  - 步骤1：下载OBS
  - 步骤2：设置录制源
  - 步骤3：选择输出格式
  - 步骤4：开始录制
- 文字：AI给出详细步骤，包括常见问题
- 旁白："OpenClau秒回！它告诉我第一步去官网下载OBS，第二步设置录制源选窗口，第三步输出格式选MP4，还有常见问题的解决办法。比看教程直观多了。"

### 第四幕：实际操作（20秒）
- 画面：按OpenClau步骤操作
  - 打开OBS
  - 设置录制源
  - 开始录制屏幕
  - 停止录制
- 文字：跟着AI步骤，一步步操作
- 旁白："我跟着OpenClau说的，一步步操作。打开OBS，添加窗口采集，选输出格式，点击开始录制。真的录上了！"

### 第五幕：成果（15秒）
- 画面：展示录制好的视频
  - 播放1-2秒
  - 表情：惊喜
- 文字：第一次成功录屏！
- 旁白："第一次成功录屏！虽然只有5秒钟，但完整的。OpenClau教我学会了录屏。"

### 第六幕：预告（10秒）
- 画面：箭头+关注
- 文字：下期：AI教我剪辑视频
- 旁白："下期视频，我用我会用AI学习怎么剪辑。关注我，一起从0到1。"

### 尾片（5秒）
- 文字：AI教我学会的，我也能学会
- 背景：频道LOGO

## 核心优势

### 1. 真实感
- 每期都是真实的"从不会到会"
- 观众能看到你的进步
- 产生共鸣

### 2. 可复制性
- 每个技能都有AI教的步骤
- 观众可以跟着学
- 降低门槛

### 3. 工具自然植入
- 不是硬推工具
- 工具是学习手段
- 成长才是目的

### 4. 内容源源不断
- 每学一个新技能 = 1期新视频
- 永远不会没内容
