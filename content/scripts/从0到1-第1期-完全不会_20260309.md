# 视频脚本 - 从0到1：我如何用AI做自媒体博主（第1期）

## 标题片（4秒）
文字：想做自媒体，但完全不会...
背景：迷茫感
音乐：略带疑问的音效

## 第一幕：现状（18秒）
- 画面：对着镜头（或手机自拍）的自述
  - 表情：略带迷茫
  - 手势：摊手、摇头
- 文字：想做自媒体博主，但完全不会
- 旁白："我做了5年Java开发，一直想做自媒体博主。但每次想到要录视频、剪辑、做封面，就放弃了。因为...我完全不会。"

## 第二幕：具体困难（25秒）
- 画面：列举困难的快速切换
  1. "我不会录屏"（摇头）
  2. "我不会剪辑"（摊手）
  3. "我不会做封面"（皱眉）
  4. "我不懂镜头语言"（看旁边）
- 文字：每一步都是拦路虎
- 旁白："我不会录屏，不会用OBS。我不会剪辑，不会用PR、剪映。我不会做封面，不会PS。甚至面对镜头，我都不知道说什么。每一步都是拦路虎。"

## 第三幕：转机（25秒）
- 画面：突然想到的表情
  - 眼神一亮
  - 微笑
- 文字：等等...这些都可以用AI做？
- 旁白："但今天我突然想到：现在是AI时代啊！我写代码都用AI了，为什么做视频不行？录屏、剪辑、封面、文案，这些不都是AI能做的吗？"

## 第四幕：决定（18秒）
- 画面：坚定的表情
  - 握拳
  - 点头
- 文字：从今天开始，我公开记录从0到1的过程
- 旁白："我决定做一件事：从今天开始，我公开记录自己从0到1做自媒体博主的过程。每学一个新技能，就做一期视频。用AI教我做自媒体。"

## 第五幕：预告（15秒）
- 画面：列出后续计划
  1. AI教我录屏
  2. AI教我剪辑
  3. AI教我做封面
  4. AI教我写脚本
- 文字：下期：AI教我录屏
- 旁白："下期视频，我会用AI学习怎么录屏。如果你也想做自媒体但完全不会，关注我，我们一起学。"

## 尾片（5秒）
- 文字：从0到1，记录真实的成长
